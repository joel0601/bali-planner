# 交互式巴厘岛行程生成问卷

本文件定义了与用户交互的完整问答流程。每个问题的答案直接影响最终行程的生成逻辑。

---

## 阶段一：你是谁？

### Q1: 同行关系
**提问**：你们是什么关系一起去巴厘岛？

| 选项 | 影响 |
|------|------|
| A. 情侣/蜜月 | 推浪漫体验（花瓣浴、悬崖日落、高端餐厅）、双人别墅 |
| B. 朋友结伴（2-4人） | 推社交活动（Beach Club、青旅、冲浪课）、性价比优先 |
| C. 带娃家庭 | 推安全区域（努沙杜瓦、金巴兰）、亲子活动、度假村 |
| D. 独自旅行 | 推青旅/社交住宿、冲浪课、数字游民体验、低价SPA |
| E. 带父母 | 推舒适节奏、包车为主、文化景点（寺庙、火舞）、避免暴晒活动 |

### Q2: 出行目的
**提问**：这次去巴厘岛主要想做什么？（可多选）

| 选项 | 匹配内容 |
|------|----------|
| A. 纯度假躺平 | Beach Club、SPA按摩、酒店内泡泳池 |
| B. 蜜月 | 乌鲁瓦图悬崖酒店、花瓣浴、浪漫晚餐 |
| C. 数字游民远程工作 | 仓古 Co-working Space、长租公寓、稳定网络区域 |
| D. 冲浪主线 | 库塔新手/仓古进阶/乌鲁瓦图高手浪点 |
| E. 瑜伽疗愈 | 乌布瑜伽课、日出瑜伽、冥想 retreat |
| F. 打卡经典景点 | 4天经典路线：佩尼达+乌布+乌鲁瓦图 |
| G. 冒险户外 | ATV、漂流、瀑布跳水、滑翔伞、潜水 |
| H. 摄影出片 | 跟拍服务、网红酒店、精灵坠崖、梯田秋千 |

### Q3: 第几次来巴厘岛？
**提问**：你之前来过巴厘岛吗？

| 选项 | 路线策略 |
|------|----------|

## 阶段二：时间与预算

### Q4: 出行天数
**提问**：打算在巴厘岛待几天？

| 选项 | 路线模板 |
|------|----------|
| 3-4 天 | **4天经典路线**：乌布 → 佩尼达 → 乌鲁瓦图 → 库塔返程 |
| 5-7 天 | **7天深度路线**：乌布2晚 + 佩尼达1晚 + 乌鲁瓦图2晚 + 水明漾1晚 |
| 8-14 天 | **15天全岛路线**：库塔 → 乌鲁瓦图 → 佩尼达+蓝梦岛 → 乌布 → 罗威纳 → 泗水火山 |
| 15 天+ | **全岛深度**：15天路线基础上增加停留时间 |
| 1 个月+ | **旅居模式**：1-2个固定 base + 周末短途探索 |
